## 簡單資訊概論及資料型別介紹
Byte 位元組
Bit 位元
1 Byte = 8 Bit
4 Byte = 32 bits
最大 2^32-1
取值範圍是 -2^32^ ~ 2^31^-1
也就是 -2147483648 ~ 2147483647

long long (int) 好長的整數
size of long long = 8
8 Byte = 64 bit
最大 2^64-1
取值範圍是 -2^64^ ~ 2^63^-1
```
C++可不可以做超過64 bit的運算？
```
